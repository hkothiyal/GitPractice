GitHub:
	Login -> Create Account
	Create GitHub repository --> url
  	Copy remote repository url from GitHub  -->  https://github.com/hkothiyal/GitPractice.git

Git:
	Install Git
	Create project folder --> Copy all the files of you rproject
	Make this directory as Git Local reporsitory  --> Initialize the repository
	Initialize the local repository:			git init
	Integrate local repositry with remote repository:	git remote add origin "remote rep url"
	Add File: 						touch <filename>
	Status of repository: 					git status
	Add one file in Index: 					git add <filename> 
	Add all Files in Index: 				git add -a

-----Now these files are ready to comit to git------------
	Commit one file:					git commit -m "Message goes here" filename
	Commit all files: 					git commit -a -m "Message goes here"
	Check logs:						git log
	End log by pressing q keyword
-----Pull file from Remote repository to local repository---------------
	Pull file from github to git				git pull origin master
	Pull file from github to already map local folder	git pull origin master --allow-unrelated-histories

---- We cannot modify files in github. Github is not to do any action, this is justa remote repository------
